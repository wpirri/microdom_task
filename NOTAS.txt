Solicitud inicial al Copilot:
Tengo de hacer un microservicio en python que luego voy a ejecutar en docker y kubernetes.
El microservicio debe hacer un POST HTTPS a un servidor determinado cada 3 segundos.

¿Querés que lo adaptemos a FastAPI con un endpoint  y métricas Prometheus, o preferís mantenerlo como un worker puro sin servidor HTTP?

Si, adaptalo a FastAPI y agregale el endpoint /health. Además el servicio se va a llamar microdom_task. Pasame todos los archivos que necesito de nuevo con estas apataciones.

Crear entorno virtual:
    python -m venv microdom

Entrar a entorno virtual:
    source microdom/bin/activate

Salir de entorno virtual
    deactivate


microdom_task/
│
├── app/
│   ├── main.py
│   ├── worker.py
│   ├── config.py
│   └── __init__.py
│
├── requirements.txt
├── Dockerfile
└── k8s-deployment.yaml

docker build -t microdom-task .

docker run -p 8081:8081 -v /var/log/microdom:/app/logs -v /etc:/app/etc microdom-task
docker run -p 8081:8081 -v /var/log/microdom:/app/logs -v /etc:/app/etc microdom-task -d --restart unless-stopped --name microdom-task
